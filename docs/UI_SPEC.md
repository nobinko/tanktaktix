# TankMatch UI仕様書（画面ごとの要素）

本書は TankMatch の UI 要素を「画面ごと」に整理した仕様書です。HUD/ミニマップ/ログ/チャット/ロビー/ルームの表示・操作要件を明文化します。

## 1. ロビー画面（Lobby）
- **タイトル/ロゴ**: ゲームタイトル、バージョン表示。
- **プレイヤー情報**: 自分の名前・アイコン・所持スキン。
- **マッチメニュー**: 「ルーム作成」「ルーム検索」「クイックマッチ」。
- **通知/ログ**: 運営通知・メンテナンス情報の表示。
- **チャット**: ロビー全体チャット（最新順、未読数バッジ）。
- **設定**: 音量、操作設定、言語切替。

## 2. ルーム画面（Room）
- **ルーム情報**: ルーム名、ルール種別（個人戦/チーム戦）、最大人数、観戦可否。
- **参加者リスト**: プレイヤー名、準備状態、チーム/色。
- **マップ選択**: マップ名、プレビュー、難易度指標。
- **ルール設定**: ターン時間、初期AP、アイテム制限。
- **準備ボタン**: Ready/Cancel。
- **チャット**: ルームチャット（参加者限定）。
- **開始ボタン**: ホストのみ表示。

## 3. バトル画面（Battle）
### 3.1 HUD
- **AP表示**: 現在AP/最大AP。
- **HP表示**: アクティブユニットのHPゲージ。
- **武器/アイテム**: 所持武器、クールダウン、残弾数。
- **ターン情報**: 現在プレイヤー、残り時間、ラウンド数。
- **状態異常**: スモーク内、バフ/デバフのアイコン表示。

### 3.2 ミニマップ
- **地形表示**: マップの全体形状。
- **ユニット位置**: 自分は明るく、敵は視界内のみ表示。
- **目的地マーカー**: AIM位置や移動先のプレビュー。
- **イベント表示**: 爆発、スモーク発生地点。

### 3.3 ログ
- **戦闘ログ**: 攻撃、回復、撃破ログ。
- **システムログ**: ターン開始/終了、ラウンド終了。
- **フィルタ**: 戦闘/システムの切替。

### 3.4 チャット
- **バトルチャット**: 全体/チーム切替。
- **クイックチャット**: 定型文（「援護求む」「狙撃中」など）。
- **ミュート**: プレイヤー個別ミュート。

## 4. 観戦画面（Spectate）
- **観戦HUD**: 観戦中のプレイヤー情報、ターン状況。
- **切替UI**: 観戦対象のユニット選択。
- **ログ/チャット**: バトルと共通。

## 5. 結果画面（Result）
- **勝敗表示**: 勝利/敗北の演出。
- **スコア**: 撃破数、被弾数、回復量。
- **リワード**: 獲得報酬、経験値。
- **再戦/退出ボタン**: 次の行動選択。
