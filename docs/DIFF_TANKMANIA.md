# TankMania 差分メモ（TankMatch基準）

本書は TankMatch の仕様を基準として、TankMania 側で確認すべき差分候補・参考リンク・未確認メモを分離して整理したものです。

## 1. 差分候補
TankMatch 仕様を前提に、TankMania の挙動やUIが異なる可能性がある項目。

### 1.1 操作/バトル
- 移動コスト（直交/斜め）の扱いが異なる可能性
- AIM 中のカメラ挙動（追従/固定/ズーム）の差異
- 攻撃後の再行動可否（AP残量と行動順）

### 1.2 ルール
- 個人戦/チーム戦の勝利条件の差異
- 観戦モードの有無、観戦可能範囲の差異
- ターン時間/ラウンド数の初期値差異

### 1.3 アイテム
- 医療アイテムの回復量/対象制限
- ボムの爆発範囲/地形破壊の有無
- ロープの到達距離/障害物無視の仕様
- スモークの視界遮断ルール（完全遮断/命中率低下）

### 1.4 UI/画面
- HUD 表示（AP/HP/武器/状態異常）の表示優先度
- ミニマップの敵表示条件（視界内のみ/常時）
- ログ/チャットの表示位置・フィルタ仕様
- ロビー/ルームの操作フロー（クイックマッチ有無）

## 3. 未確認メモ
- TankMania のターン進行が「同時行動」か「順番制」か要確認

- 視界/索敵ルール（Fog of War）の有無
- チーム戦での味方視界共有の有無
