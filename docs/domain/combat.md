## クールダウン

- 1カウント = 300ms
- カウント中はすべての行動（移動・射撃）が不能（移動予約のみ受け付ける）
- 移動クールダウンは到着時に発動（移動中は `isMoving` で管理）
- **短距離**: 移動距離 < 200px → 1500ms / 5カウント
- **長距離**: 移動距離 ≥ 200px → 2100ms / 7カウント
- 距離判定: 移動元（最後の予約地点 or 現在地）から移動先までの直線距離

### UI表示

| 状態 | 表示 |
|---|---|
| 移動中（`isMoving === true`） | カウント非表示 |
| 到着後 / 射撃後 | 即座にカウントダウン開始（300ms ごとに1減少） |
| カウント = 0 | 次のアクションが可能 |

## AIM（スリングショット方式）

1. 自分のタンク上でマウスボタン押下 → AIMモード開始
2. ドラッグ中: ドラッグ方向と**逆方向**に射撃ガイドライン表示
3. 自機の半径（18px）以内に戻してボタンを離す → **キャンセル**
4. 半径外でボタンを離す → **発射**
   - 発射方向 = ドラッグベクトルの逆方向（単位ベクトル）

## 射撃条件（サーバ検証）

| 条件 | 結果 |
|---|---|
| クールダウン中 | 拒否 |
| 移動中（`isMoving === true`） | 拒否 |
| リスポーン無敵期間中 | 拒否 |
| 残弾 0 かつ bomb 非所持 | 拒否 |
| bomb 所持かつ残弾 0 | 発射可（弾薬消費なし） |

---

## 命中判定

- アルゴリズム: Liang-Barsky（弾丸をタンクのローカル座標系に変換して AABB チェック）
- 対象形状: タンクの回転矩形（26×20px）
- 壁衝突・マップ外到達で爆発

---

## HP / 弾薬

| パラメータ | 初期値 | 変化 |
|---|---|---|
| HP | 100 | 被弾ごとに 20 減少。0 で即死 |
| 弾薬 | 20 発 | 射撃ごとに -1。最大 40（ammo/ammoPass で回復） |

instantKill モード時: HP 最大値は 20（1発即死）。

## フレンドリーファイア

| ケース | 結果 |
|---|---|
| 自爆 | ダメージあり |
| 味方への攻撃 | ダメージなし（弾丸は貫通扱い） |
| 敵への攻撃 | 通常ダメージ |

---

## リスポーン

- HP 0 → 即座にリスポーン地点へ移動（インスタントリスポーン）
- 無敵期間: 1500ms（`RESPAWN_COOLDOWN_MS`）
- 無敵期間中: 被弾なし・行動不可・クライアント描画は α=0.5（半透明）
- リスポーン地点: チームスポーンポイント ±40px のランダム範囲、最大20回試行、全失敗時はスポーンポイント初期座標にフォールバック
- スコア・キル・デス: リスポーン後も維持（初回参加時のみリセット）

---

## 制約

- ダメージ計算はサーバのみ。クライアントは表示のみ
- 爆発は即時ブロードキャスト（tick を待たない）
- フレンドリーファイアは絶対に発生しない（サーバが拒否）
- bomb 発射時は弾薬を消費しない
