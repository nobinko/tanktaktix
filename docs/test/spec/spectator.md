# テスト仕様：観戦モード (Spectator Mode / B-6)

## 概要
ルームで行われている試合をプレイヤーとしてではなく観戦者として視聴できるモード。
観戦者はチームに所属せず、全プレイヤーの位置を隠密状態を含め常に表示する。

## 検証項目

### 1. 観戦参加
- **spectateRoom メッセージ**: `spectateRoom` メッセージ送信で観戦者としてルームに参加できるか。
- **ルーム状態受信**: 参加後に `room` メッセージを受信し、ルーム名・マップデータ等が含まれるか。
- **パスワード保護**: パスワード付きルームで正しいパスワードが要求されるか。

### 2. 可視性（全プレイヤー表示）
- **プレイヤー一覧**: 観戦者の `room` メッセージに全プレイヤー情報が含まれるか。
- **隠密無視**: ブッシュ内のプレイヤーも観戦者には可視であるか。
- **自身の除外**: 観戦者自身はプレイヤー一覧に含まれないこと。

### 3. アクション制限
- **移動不可**: `move` メッセージ送信が無視されるか。
- **射撃不可**: `shoot` メッセージ送信が無視されるか。
- **移動キャンセル不可**: `moveCancelOne` がプレイヤー一覧に影響しないか。

### 4. プレイヤースロット
- **枠非消費**: 観戦者が `maxPlayers` 枠を消費しないこと。
- **満員でも参加可**: ルームがプレイヤーで満員でも `spectateRoom` で参加できるか。
- **複数観戦者**: 複数の観戦者が同時に参加できるか。

### 5. チャット
- **送信可能**: 観戦者がチャットメッセージを送信できるか。
- **名前区別**: 観戦者のチャットに `👁` プレフィックスが付与されるか。
- **受信可能**: ルーム内のチャットメッセージが観戦者にも届くか。

### 6. 退出・ライフサイクル
- **ロビー帰還**: `leaveRoom` でロビーに戻れるか。
- **spectatorCount 更新**: 退出後にロビー一覧の `spectatorCount` が正しく減少するか。
- **ルーム削除判定**: プレイヤーと観戦者の両方が0になるまでルームが削除されないこと。
- **ゲーム終了通知**: ゲーム終了時に `gameEnd` メッセージが観戦者にも送信されるか。

### 7. クライアントUI
- **Watch ボタン**: ロビーのルーム一覧に Watch ボタンが表示されるか。
- **SPECTATING バッジ**: HUD に「📺 SPECTATING」が表示されるか。
- **入力無効化**: マウスクリック（移動/射撃）が無効化されるか。
- **カメラ操作**: 矢印キー・ズーム・回転が正常に動作するか。
- **カメラ初期位置**: 観戦者のカメラがマップ中央から開始するか。
