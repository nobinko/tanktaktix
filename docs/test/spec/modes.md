# テスト仕様：ゲームモード (Game Modes - CTF)

## 概要
CTF (Capture The Flag) モード特有のルール、スコアリング、およびオブジェクトの状態遷移を検証します。

## 検証項目

### 1. 旗の基本ルール
- **取得**: 敵旗に接近 (25px + 18px) することでキャリア状態になるか。
- **輸送**: キャリアーの頭上に 🚩 アイコンが表示され、移動に合わせて旗が追従するか。
- **回収**: 自陣の旗を奪われている際、旗を持たない味方が敵が持っていない自旗に触れると即座にベースに戻るか。

### 2. 得点判定
- **得点条件**: 敵旗を保持した状態で **自陣のスポーンゾーン (200x200)** 内で **完全停止** (`isMoving === false`) すること。
- **スコア加算**: チームスコア +5、個人ボーナス +5 が正しく記録されるか。
- **旗のリセット**: 得点後、両チームの旗がベース位置にリセットされるか。

### 3. ドロップ・ロスト
- **ダメージドロップ**: キャリアーがダメージを受けた際、旗がベース位置（`flagPositions` の初期位置）に即時リセットされるか。
- **死亡ドロップ**: キャリアーが撃破された際、旗がベース位置（`flagPositions` の初期位置）に即時リセットされるか。

## 関連 acceptance criteria
