# テスト仕様：基本メカニクス (Core Mechanics)

## 概要
タンクの基本移動、旋回、射撃、およびそれに付随するリソース管理とクールダウンの正確性を検証します。

## 検証項目

### 1. 移動メカニクス
- **移動距離制限**: 一度のクリックによる移動距離が `MAX_MOVE_DIST(300px)` でクランプされるか。
- **ピボットターン**: 車体が目標方向を向くまで前進が開始されないこと。
- **移動予約**: `moveQueue` (最大5件) への追加と順次実行。
- **キャンセル (Zキー)**: 予約リストの末尾が正しく削除されるか。

### 2. 戦闘メカニクス
- **射撃**: スリングショット方式による発射方向の正確性。
- **弾丸飛翔**: 弾速 `220px/s` および射程 `99999px`（無制限）の維持。
- **判定**: `Liang-Barsky` アルゴリズムによる回転矩形への命中判定。

### 3. ステータス管理
- **HP**: 被弾ごとの 20% 減少、および 0% での即時死亡。
- **弾薬**: 射撃ごとの消費と、アイテムによる回復 (上限40発)。
- **クールダウン**:
    - ショートムーブ (< 200px): 1500ms
    - ロングムーブ (>= 200px): 2100ms
    - 射撃: 1800ms
- **リスポーン無敵**: 1500ms の無敵時間と視覚的表現 (半透明)。
