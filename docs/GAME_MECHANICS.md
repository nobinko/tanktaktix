# ゲームメカニクス仕様書: クールダウンと移動

このドキュメントは、戦車の移動および射撃に関するクールダウン（硬直時間）の仕様を定義するものです。これらはゲームバランスの核心となる要素です。

## クールダウンの基本単位
すべてのクールダウン時間は「カウント」という単位で管理・表示されます。
- **1カウント = 0.3秒 (300ms)**
- クライアント（画面上）では、残りカウント数が戦車の頭上に数値で表示されます。

## アクション別クールダウン仕様

### 1. 移動 (Movement)
移動距離に応じて、発生する硬直時間が変動します。

| アクション | 条件 (移動距離) | クールダウン時間 | カウント表示 | 備考 |
| :--- | :--- | :--- | :--- | :--- |
| **ショートムーブ** | **200未満** (全長の約2/3未満) | **1.5秒** | **5** から開始 | 小回りが利く移動。 |
| **ロングムーブ** | **200以上** (全長の約2/3以上) | **2.1秒** | **7** から開始 | 大きく移動する際のリスク。 |

*   **距離判定**: プレイヤーがクリックした地点までの直線距離（壁や最大移動距離による補正後の確定距離）に基づきます。
*   **最大移動距離**: 1回の移動指示で移動できる最大距離は **300** です。

### 2. 射撃 (Shooting)
射撃を行った際に発生する硬直時間は固定です。

| アクション | クールダウン時間 | カウント表示 | 備考 |
| :--- | :--- | :--- | :--- |
| **射撃** | **1.8秒** | **6** から開始 | 標準的な攻撃後の硬直。 |

---

## 補足: UI表示仕様
- **移動中**: クールダウンカウントは表示されません（移動中はアクション不能）。
- **到着後 / 射撃後**: 即座にカウントダウンが開始されます。
- **カウントダウン**: 0.3秒ごとに数値が1減ります（例: 7 → 6 → 5 ... → 0）。
- **0になった瞬間**: 再び次のアクション（移動または射撃）が可能になります。
